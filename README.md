## 项目简介
这套模板作为项目常用UI布局，分为两种：   
1、地图作为模块被框起来的布局，对应文件Home.vue   
2、地图作为背景的布局，对应文件Second.vue   
首页的内容元素分为8个模块框，以组件的方式引入，类别有：图表、文字滚动、静态文字展示。同时支持不同分辨率的展示。
登录页也做了常规的布局：大背景图+登录框。
